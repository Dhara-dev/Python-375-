This project focuses on analyzing crime data from 2020 to 2024 using Python. 
The objective is to extract meaningful insights, identify patterns, and build a predictive model using machine learning techniques.

Key Features:
- Data Cleaning and Preprocessing using Pandas
- Exploratory Data Analysis (EDA) with 8+ visualizations
- Crime trend analysis over time
- Correlation analysis between different variables
- Machine Learning model (Linear Regression) for prediction
- Statistical hypothesis testing (Z-test)

Tools & Technologies:
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (Visualization)
- Scikit-learn (Machine Learning)
- SciPy (Statistical Testing)

Insights:
- Identified patterns in crime frequency over years
- Observed relationships between different crime attributes
- Evaluated model performance using Mean Squared Error (MSE)

This project demonstrates the practical application of data science concepts including EDA, statistical analysis, and machine learning.
